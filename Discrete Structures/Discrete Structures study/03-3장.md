- 문제 2 : A={1,2,3,4,5} 일 때, {1,3} ∈ A x A

    - **False**
 
    - A x A는 데카르트 곱(Cartesian Product) : 순서쌍 (a, b)로 이루어진 집합
 
    - 즉, A × A = {(1,1), (1,2), …, (5,5)}
 
    - 따라서 {1,3} 이 아니라 **{(1,3)}** 와 같이 **순서쌍**으로 표현해야함

<br/>

- 문제 3 : A = {1,2,3,4,5} 일 때, ∅ ∈ P(A)

    - **True**
 
    - P(A)는 A의 멱집합(Power Set) : 즉 A의 모든 부분집합들의 집합
 
    - **멱집합에는 반드시 공집합 ∅도 포함**

<br/>

### cardinality (크기)

- 집합 A와 집합 B가 있다고 하자

    - 두 집합의 크기가 같을 때 : |A| = |B|
 
        - **일대일 대응**일 때만 성립 
 
        - ex: f(n) = 2n 이고 무한 집합의 경우
     
            - 1 → 2,

            - 2 → 4,

            - 3 → 6, . . .
         
        - ex: 유한집합인 경우
        
            - A = {사과, 바나나, 포도}
            
            - B = {빨강, 파랑, 초록}   

    - B 집합이 A 집합보다 크거나 같을 때 : |A| ≤ |B|
 
        - **일대일 함수** 일 때 성립

### countable set (가산 집합) & 비가산 집합(uncountable set)

- **countable set (가산 집합)** : 집합이 유한하거나, 양의 정수 집합(Z⁺)과 크기(기수)가 같을 때

    -  양의 정수 집합(Z⁺)과 크기(기수)가 같다는 것은 해당 집합의 원소를 1,2,3 .. 처럼 셀 수 있다는 것이다
 
        - 자연수 집합 : Z⁺ = {1, 2, 3, 4, …}

            - 자기 자신과 1대1 대응 가능

        - 짝수 집합 : E = {2, 4, 6, 8, …}
          
            - f(n) = 2n 으로 자연수와 1대1 대응 가능

        - 정수 전체 집합 : Z = {0, 1, −1, 2, −2, 3, −3, …}
          
            - 자연수로 순서를 지그재그로 붙이면 1대1 대응 가능

        - 유리수 집합 : Q = {p/q | p, q ∈ Z, q ≠ 0}
          
            - 표처럼 배열해 “대각선 방법”으로 하나씩 세면 셀 수 있음
         
    - 가산 집합을 **증명**하기 위해선 **일대일대응**을 보이면 된다
 
        - 일대일 대응이라는 것은 매칭이 되어 순서를 매길 수 있다는 의미
     
        - 만약 [0,1]과 같은 비가산 집합에서는 아무리 나열하려 해도 빠뜨리는 원소가 존재함 

<br/>

- **비가산 집합(uncountable set)** : 셀 수 없는 집합

    - 실수 집합 R
       
        - 0과 1 사이 구간 (0,1)조차도 비가산 집합 (Cantor의 대각선 논법으로 증명)

    - 구간 집합 [0,1]
    
        - 이 구간 안에도 무한히 많은 실수가 있으므로 자연수로는 전부 나열 불가

<br/>

- **핵심은 무한집합이라고 해서 무조건 비가산 집합이 아니다**

<br/>


### 가산집합 예시 문제 1

- 문제 : 양의 짝수 집합 E가 가산집합임을 보여라

    - 함수 f(x) = 2x라고 정의하자
 
    - 자연수 1,2,3,… 하나하나가 정확히 짝수 2,4,6,…와 연결된다
 
    - 그러면 f는 Z+(양의 정수)에서 E (양의 짝수)로 가는 **전단사(bijection)** 이다
 
        - 일대일임을 보이기 위해, f(n) = f(m)이라고 가정하자. 그러면 2n = 2m이 되고, n = m 이므로 다른 자연수를 넣으면 절대 같은 짝수가 나오지 않는다
     
        - 전사를 증명하기 위해 t가 양의 짝수라고 하자. 그러면 t=2k인 양의 정수 k가 존재하고, f(k)=t이다
     
            - 즉, 임의의 짝수 t는 반드시 어떤 자연수 k를 2배를 했을 때 나온다
