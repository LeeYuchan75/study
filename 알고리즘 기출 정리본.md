### DP 알고리즘 

```ruby
def max_partial_sum(a):
    n = len(a)
    if n == 0:
        return 0
    if n == 1:
        return a[0]

    Ai = [0] * n
    Ai[0] = a[0]
    Ai[1] = max(a[0], a[1])

    for i in range(2, n):
        Ai[i] = max(Ai[i-1], Ai[i-2] + a[i])

    return Ai[-1]
```
