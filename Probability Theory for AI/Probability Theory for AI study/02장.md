### 집합론(Set Theory)의 기초 개념

- **집합**

    - **순서가 없는** 원소들의 모임이다
 
        - {1, 2, 3} 과 {3, 2, 1} 는 같음

- **집합의 예시**

- A = {♣, ◇}
    - 집합 A에는 ♣와 ◇ 두 원소가 들어있다

- ◇ ∈ A ; ♡ ∉ A
    - ◇는 A에 포함된다, ♡는 A에 포함되지 않는다

- B = {1, 2, 3}
    -  1, 2, 3이라는 세 숫자의 집합이다

- C = {x² : x = 1, 2, 3} = {1, 4, 9}
    - 집합 C는 1,2,3을 제곱한 값들을 원소로 가진 집합이다

- D = {H, T}
    - D는 동전 던지기의 결과(앞면 H, 뒷면 T) 집합이다

<br/>

- **특별한 집합 기호**

    - 자연수 집합, **N** = {1, 2, 3, …}
        - 1부터 시작하는 수.

    - 정수 집합, **Z** = {…, −3, −2, −1, 0, 1, 2, 3, …}
        - 음수, 0, 양수를 모두 포함.

    - 실수 집합, **R**
        - 소수, 무리수까지 포함하는 모든 수.

<br/>

- **부분집합 (Subset)**

    - **A ⊂ B ≡ (x ∈ A ⇒ x ∈ B)** 

    - 집합 A의 모든 원소가 B에도 들어있으면 A는 B의 부분집합
 
    - ex: N ⊂ Z
 
        - 모든 자연수는 정수의 부분집합 

<br/>

- **집합의 같음 (Equality)**

    - **A = B**

    - 두 집합이 서로 부분집합 관계이면, 두 집합은 같다
 
    - ex: {a,a,b} = {a,b}

        - 중복된 원소는 무시되므로 동일하다

<br/>

- **전체집합 (Universal set)**

    - S 또는 Ω로 표기

- **공집합 (Null set)**
    
    - ∅ 또는 {}
 
    - **공집합은 모든 집합의 부분집합**

<br/>

- **Venn Diagrams**

    - 집합을 닫힌 도형(보통 원, 사각형 등)으로 나타낸 것
 
    - ∪ : 합집합 (Union)
 
    - ∩ : 교집합 (Intersection)
 
    - Aᶜ 또는 Ā : 여집합 (Complement)
 
    - A − B 또는 A ∩ Bᶜ : 차집합 (Difference)
 
    - A₁, A₂, …, Aₙ are m.e. if Aᵢ ∩ Aⱼ = ∅, i ≠ j : 서로소 집합 (Mutually Exclusive, Disjoint)
 
        - 서로소 : 집합 A와 B의 교집합이 공집합, 각 원이 떨어져있는 구조 


    - A₁ ∪ A₂ ∪ A₃ ∪ … ∪ Aₙ = S : 분할 (Partition)

        - 모든 부분집합들을 합치면 전체집합 S가 되어야 한다

        - 각 부분집합들은 서로소여야함. 즉 겹치지 않아야 함

<br/>

### **De Morgan's Law (드모르간 법칙)**

- A와 B의 합집합의 여집합은 A의 여집합과 B의 여집합의 교집합과 같다

![System Resources](../../images/Probability%20Theory%20for%20AI%20images/드모르간법칙.png)

<br/>

- 예시 문제 : S = {1,2,3,4,5,6}, A = {1,2}, B = {2,4,6} 일 때 다음을 구하라

    -  A ∪ B = {1,2,4,6} (둘 중 하나라도 포함)

    -  A ∩ B = {2} (공통 원소)

    -  Aᶜ = {3,4,5,6} (S 안에서 A를 뺀 나머지)

    -  Bᶜ = {1,3,5}

    -  (A ∪ B)ᶜ = {3,5}

    -  Aᶜ ∩ Bᶜ = {3,5} (드모르간 법칙 확인)

<br/>

### Distributive Law (분배법칙)

- A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)

    - A와 (B와 C의 교집합)의 합집합은, (A와 B의 합집합)과 (A와 C의 합집합)의 교집합과 같다 

<br/>

### Functions (함수의 정의)

![System Resources](../../images/Probability%20Theory%20for%20AI%20images/함수설명.png)

- f : X → Y

     - 함수 f는 집합 X에서 집합 Y로 가는 대응 관계이다

- X : Domain

     - 정의역(Domain): 함수에 입력할 수 있는 값들의 집합

- Y : Co-domain

    - 공역(Co-domain): 함수가 도달할 수 있는 "가능한 출력값들의 집합" (전체 범위)

- ∀ x ∈ X, f(x) ∈ Y

     - 모든 x가 X에 속하면, f(x)는 Y에 속한다
 
         - 실제로 함수가 만들어내는 출력값들의 집합. 공역 Y의 부분집합이 될 수 있다 

- 일대일 함수
  
    - f(x1​) = f(x2​) ⇒ x1 ​= x2​ : 

    - 역함수가 존재하는 함수, 서로 다른 입력값이 항상 서로 다른 출력값을 만든다
 
    - 모든 출력값(y값)이 반드시 사용될 필요는 없다

- 일대일 대응

    - 일대일 함수(injective) + 전사(surjective)
 
    - 일대일 함수를 만족하며, 모든 출력값이 정확히 하나의 입력값과 짝지어진다
      
<br/>

- 만약 함수 f를 x²로 정의한다면

    -  정의역도 실수 전체, 공역도 실수 전체
 
    -  하지만 실제 Y값 범위는 음수가 될 수 없으므로 0 이상인 값들로 영역을 정할 수 있음

### Countable and Uncountable Sets (가산집합과 비가산집합)

- 집합 A의 크기(**카디널리티**)

    - A에 있는 **원소의 개수** 라는 의미로 **|A|** 로 표기
 
    - 원소 개수가 유한하면 **유한 집합**이다
 
        - |A| < ∞

- **가산집합 (countable set)**

    - 집합이 **유한**하거나, 원소들을 1,2,3,…처럼 순서대로 **셀 수 있는**집합
 
    - 유한집합이든 무한집합이든 셀 수 있으면 가산집합임
 
        - ex: 자연수 집합 N은 무한하지만, 1,2,3,…처럼 셀 수 있으므로 가산집합

![System Resources](../../images/Probability%20Theory%20for%20AI%20images/가산집합공식.png)

<br/>

- **비가산집합 (Uncountable Sets)**

    - **셀 수 없는** 집합
 
        - ex: 실수 전체 ℝ, 구간 [0,1] 같은 집합은 비가산집합
           
<br/>

### Random Experiment (확률 실험)

- 용어 정리

    - **Random Experiment** : 결과를 확실히 예측할 수 없는 현상
 
        - Roll a die (주사위를 한 번 던지기)

        - Roll a die three times (주사위를 세 번 던지기)

        - Flip a coin (동전 던지기)

    - **Outcome (결과)** : 무작위 실험을 실제로 했을 때 나온 값
 
        - 집합의 원소(Element)로 표현 
 
        - Roll a die → 결과 = 3

        - Roll a die three times → 결과 = (2, 3, 6)

    - **Events (사건)** : 여러 가능한 결과들의 모임
 
        - 집합(Set)으로 표현

        - E1 = {1, 3, 5} : 주사위 눈이 1 또는 3 또는 5가 나오는 사건 

        - E2 = {2, 4} : 주사위 눈이 2 또는 4가 나오는 사건 

        - E3 = {6} : 주사위 눈이 6이 나오는 사건 
     
    - **Partition (분할)** : 체를 빠짐없이 포함하고, 동시에 서로 겹치지 않는 사건들의 집합
 
        - B₁, B₂, …, Bₙ이 Partition이 되려면 B₁, B₂, B₃, …, Bₙ의 합집합은 표본공간 Ω와 같다


    - **Sample Space (표본공간, Ω)** : 무작위 실험에서 일어날 수 있는 모든 결과들의 집합
 
        - 전체집합(Universal set)으로 표현

        - Roll a die → Ω = {1, 2, 3, 4, 5, 6}

        - Roll a die three times → Ω = {(1,1,1), (1,1,2), … , (6,6,6)}


    - 쉽게 정리
 
        - 표본공간 = 전체 경우의 수

        - 사건 = 그 안에서 관심 있는 부분집합

        - 결과 = 실제로 나온 하나의 값
     
        - 분할 (Partition) = 표본공간을 빠짐없이 겹치지 않게 나눈 작은 조각들의 집합

        - 결과가 사건에 속하면 → 그 사건이 발생한 것

<br/>

### 주사위 예시

- 주사위를 던지면 표본공간은 {1, 2, 3, 4, 5, 6}이다

    - 주사위를 던졌을 때, 사건 E₁은 {1, 2}, 사건 E₂는 {4}라고 가정하자

            - 즉 , 6 x 6 x 6 = 216개의 경우가 존재한다

### Probability (확률 정의)

- 사건 A에 대해, 그 사건이 일어날 확률을 **P(A)** 라고 한다

    - P(A)는 **실험을 아주 많이 반복했을 때** 사건 A가 관측되는 비율이다
 
        - 확률은 이론적 개념이지만, 직관적으로는 "많이 반복했을 때 사건이 나타나는 빈도"로 이해할 수 있다 

![System Resources](../../images/Probability%20Theory%20for%20AI%20images/확률사건그림.png)

- **Axioms of Probability (확률 공리)**

    - **P(A) ≥ 0** : 어떤 사건 A라도 확률은 0보다 크거나 같다
 
    - **P(S) = 1** : 표본공간 S 전체의 확률은 1이다
 
    - 서로 겹치지 않는(disjoint) 사건 A1, A2, A3,…에 대해 **P(A1 ∪ A2 ∪ A3 ⋯) = P(A1) + P(A2) + P(A3) + ⋯** 이다 

<br/>

- **주사위 예시**

- P({1}) = P({2}) = ⋯ = P({6}) 라 가정

    - 1 또는 2 또는 … 또는 6이 나온다”의 확률은 각각 사건의 확률 합과 같다 -> 즉 1
 
    - 전체 표본공간의 확률은 1이다

### Countable Sample Space (셀 수 있는 표본공)

- **이산 확률공간**-
    
    - 셀 수 있는 경우들로 이루어진 공간을 말함
 
    - 만약 셀 수 있는 집 A = {a₁, a₂, a₃, …} 가 존재한다면
     
        - P(A) = P(a₁) + P(a₂) + P(a₃) + … 로 사용 가능 
     
        - 하지만 **연속적인 공간에서는 사용 불가능**
     
            - 연속 공간에서는 적분을 사용함  

<br/>

- **연속 확률공간**

    - 표본공간을 **셀 수 없을** 때 

    - S = ℝ⁺ = {x ∈ ℝ, x ≥ 0} : 실수 전체 중에서 0 이상인 값들
 
    - 만약 [0,1] 구간이 존재하고 여기 무작위로 한 점을 선택한다고 하자
 
        - P([0,0.5]) = 0.5
     
        - P([0,0.25]) = 0.25
     
        - 즉, **P([a,b]) = b – a, 0 ≤ a ≤ b ≤ 1** 라는 공식이 도출되고
     
        - 전체 길이가 1이 아니라 L로 일반화하면 아래의 공식이 도출된다
     
        - ![System Resources](../../images/Probability%20Theory%20for%20AI%20images/연속공간공식.png)
        - 따라서 P([0.5,0.5]) = 0.5 – 0.5 = 0 이기 때문에 **연속 공간에서 한 점의 확률은 0 이다**

- 문제 : 어떤 기계의 수명 T가 x년 이상일 확률이 1/(2^x)라고 주어졌다

- (1) : P(T>=1)  
    
    - x=1 대입 → 1/2^1 = 1/2  

- (2) : P(T>=2)  

    - x=2 대입 → 1/2^2 = 1/4  

- (c) P(1<=T<=2)  

    - **P(T>=1) - P(T>=2) = 1/2 - 1/4 = 1/4**

### 베이즈 정리 & 조건부확률 차이점 

- **조건부확률**

    - B가 일어났다는 사실을 알고 있을 때, A가 일어날 확률을 계산한 것
    
    - 즉, **원인**이 발생했을 때 **결과가 나온 확률**을 구한 것이다
 
    - ex: 나이가 20세, 키 180cm인 사람이 농구 선수일 확률은?
 
        - 증거 -> 결과
     
        - feature가 주어졌을 때 결과를 예측하는데 사용

<br/>

- **베이즈 정리**

    - ![System Resources](../../images/Probability%20Theory%20for%20AI%20images/베이즈정리공식.png) 

    - **결과**가 나왔을 때, 해당 결과가 도출된 **원인의 확률**을 구한 것이다
 
    - ex: 검사에서 양성이 나왔다. 이 사람이 실제로 병에 걸렸을 확률은 ?
 
        - 해당 예시에서 우리가 쉽게 알 수 있는 것이
     
            - P(양성∣병 있음)
         
            - P(병 있음)
         
            - P(양성)
         
            - 이라면, P(원인|결과) = P(병 있음|양성)을 베이즈 추론을 통해 쉽게 구할 수 있다 

- 정리하면 

    - 조건부확률 = 원인 -> 결과

    - 베이즈 정리 = 결과 -> 원인
